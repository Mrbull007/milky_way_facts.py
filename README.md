# 🌌 Milky Way Galaxy Random Facts Explorer

An interactive Python console app that delivers fascinating random facts about our home galaxy, the Milky Way.

## Features
- **Random Fact Generator**: Press a key for a new surprising fact.
- **View All Facts**: Browse the complete collection.
- **Simple & Lightweight**: No dependencies beyond Python's standard library.
- Educational and fun for all ages!

## Quick Start
```bash
# Clone or download the script
python milky_way_facts.py
